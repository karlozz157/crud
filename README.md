# Crud

### Clonar proyecto
```
git clone git@github.com:karlozz157/crud.git 
```

### Instalar dependencias
```
cd crud && composer install
```

### Crear base de datos
```
symfony console doctrine:schema:create
```

### Correr servidor
```
symfony server:start
```
